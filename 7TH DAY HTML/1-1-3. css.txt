ATADIKTAN SONRA DA CSS DOSYASININ ICERISINDE BU TAGLARI CLASSLARINA GORE CAGIRARAK

.important-text{
    font-size: 40px;
    color:red;
    text-align: right;
}

.important-paragraph{
    font-size: 20px;
    color:blue;
    text-align: left;
}
