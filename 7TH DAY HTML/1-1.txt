WHAT IS HTML?
-> HYPER TEXT MARKUP LANGUAGE 
IT IS NOT PROMING LANGUAGE 
IT CAN HELP TO CREATE WEBPAGE


-> <html> </html> = BETWEEN HTML TAGS WE INSERT OUR
    HTML CODES
-> INSIDE HTML FILE THERE IS TREE SPACE
    - HEAD
    - BODY
    - FOOTER

    - HEAD : <head> </head> WE CAN WRITE THE TOP TOPICS WHICH NORMAL 
      WEBPAGE LIKE COURSES, ABOUT US, BLOG, CONTACTS, PROFESSIONS..ETC

      * <title> </title> WE NEED TO INSERT THE WEPAGE NAME INSIDE THE HEAD TAGS IN THE TITLE TAGS

    - BODY : <body> </body> MIDDLE OF THE PAGE INFORMATION
    
    * <h1> Hello </h1> we insert inside the body tags

    - FOOTER : BUTTOM INFORMATION LIKE :CLAIMS , CONTACT US ... ETC
